EC2 Instance Setup and Web Server Installation

Step-by-Step Guide

  1. Login to your AWS account.
  2. Search for EC2 in the AWS Management Console search bar.
  3. Create a new EC2 instance:
    • Give a name to your instance.
    • Select an AMI, such as Amazon Linux or Ubuntu.
    • Create a new key pair and download it.
    • Launch the instance.
  4. Connect to your EC2 instance using EC2 Instance Connect:
    • Click the Connect button in the EC2 console.
    • Select EC2 Instance Connect.

Terminal Commands

Follow these commands to install a web server on your EC2 instance:

# 1. Switch to Root User
sudo su -

# 2. Update the system
yum update -y

# 3. Install the web server (HTTPD)
yum install -y httpd

# 4. Check the status of the web server
systemctl status httpd

# 5. Create a directory and go to it
mkdir temp && cd temp

# 6. Download website template
wget https://www.free-css.com/assets/files/free-css-templates/download/page296/listrace.zip

# 7. Verify the downloaded file
ls -lrt

# 8. Unzip the downloaded file
unzip listrace.zip

# 9. List the unzipped content
ls -lrt

# 10. Navigate to the folder without the zip
cd [FolderWithoutZip]

# 11. Move the files to the web directory
mv * /var/www/html/

# 12. Go to the web directory
cd /var/www/html/

# 13. List the files in the web directory
ls -lrt

Configure Security Groups

  1. Go to the Security tab in the EC2 instance dashboard.
  2. Add new inbound rules for:
    • HTTP (Port 80)
    • HTTPS (Port 443)

Final Web Server Configuration

# 14. Check the web server status
systemctl status httpd

# 15. Enable the web server to start on boot
systemctl enable httpd

# 16. Start the web server
systemctl start httpd

# 17. Verify the web server status
systemctl status httpd

Reload the public IP address of your EC2 instance in the browser to see the deployed website!
